Sobre o Sistema

Tecnologias Usadas no Sistema.

  • LS Customs - v 1.5

    Projeto DAC
  • Back-end
    • - Java Server Faces usado nas páginas web do sistema.
    • - Enterprise JavaBeans usado em algumas classes da lógica de negócio.
    • - Java Persistence API + Hibernate usado para a persistência dos dados no banco.
    • - Java Authentication and Authorization Service (API) usado para a funcionalidade de autenticação do cliente e do administrador no sistema.
    • - Caelum Stella (API) usada para validar e formatar CPFs e CNPJ.
    • - Joda Time (API) usada para trabalhar com datas.
    • - Dependency Injection (CDI).
    • - Banco de Dados usado foi o Postgres.
  • Front-end
    • - Materialize foi usado no Front-end do sistema.
    • - Demo Materialize foi usado no Front-end do sistema.
    • - PRIMEFACES foi usado no Front-end do sistema.
